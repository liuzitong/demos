QxPack.indcom is a utils package based on QT5.

1. directorys

[ common ] -- std c++11, qxiccore package
  this folder is not depends on QT. 

[ geometry ] -- std c++11, qxiccore package
  this folder is not depends on QT.

[ afw ] -- QT5, qxicqt5 package
  application framework, used for a QT5 GUI application

[ net ] -- QT5, qxicqt5 package
  network server and client 

[ sys ] -- QT5, qxicqt5 package
  some utils classes.

[ im_algor ] -- QT5, some image algorithm

[ ui_qml_base ] -- QT5, qxicqt5ui package, QML base
  application QML base classes

[ ui_qml_control ] -- QT5, qxicqt5ui package, QML control
  application QML controls


2. mkspecs
  In the folder, contains qxiccore.pro, qxicqt5.pro, qxicqt5ui.pro, ...
